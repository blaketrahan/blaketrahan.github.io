REPLACE

Use replace to replace an existing piece of HTML with another.
There are two versions:

・replace('SomeElementID', './html/woman.html')
・replace(this, './html/woman.html')

replace ( , )
Using an element's ID will replace that element.
Using the keyword this will replace the element itself.

CREATE

Use create to create new HTML, only once.
There are two versions:

・create(this, 'SomeElementID', './html/church.html')
・create(this, null, './html/church.html')

create ( this, , )
Using an element's ID will create the new HTML inside the element.
Using null will create the new HTML at the bottom of the <body> tag.


In both cases, after the element is clicked, the classname .clicked is added to it. That way you can modify it with CSS to change the pointer to disabled, hide it, etc.

*Always pass the keyword this.
